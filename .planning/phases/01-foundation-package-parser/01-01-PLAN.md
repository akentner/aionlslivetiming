**src/aionlslivetiming/cli/jsonl_logger.py** — per D-07 and PROJECT.md §Context reverse-engineering notes:
- Public function `async def run(event_id: str, output_path: str | pathlib.Path, *, host: str = "wss://livetiming.azurewebsites.net/", max_seconds: float | None = None) -> int`:
  - Import `websockets` (top-of-module import inside `run` so test mocking can patch `aionlslivetiming.cli.jsonl_logger.websockets`).
  - Use `import orjson` with fallback to `import json` if ImportError (D-10 — orjson is optional).
  - Connect via `async with websockets.connect(f"{host}/api/livetiming", open_timeout=10, ping_interval=20, ping_timeout=20) as ws:` — note: the exact path suffix is reverse-engineered from PROJECT.md; if the user knows the actual handshake URL is different, they override via `--host` flag.
  - Send handshake JSON: `await ws.send(orjson.dumps({"eventId": event_id, "type": "INIT"}))` — match the JS bundle's `onopen` send per PROJECT.md.
  - Loop:
    ```python
    raw = await ws.recv()  # bytes or str
    payload = orjson.loads(raw) if isinstance(raw, (bytes, bytearray)) else json.loads(raw)
    line = {"ts_recv_ms": int(time.monotonic_ns() // 1_000_000) + 1_700_000_000_000, "raw": payload}
    fh.write(orjson.dumps(line).decode() + "\n")
    ```
    Use `time.time()` for `ts_recv_ms` (Unix epoch ms) not monotonic — this is wall-clock so the captured log lines up with downstream tools. The `time.monotonic_ns()` example above is wrong for this use case; use `int(time.time() * 1000)`.
  - Respect `max_seconds`: if set, stop after N seconds.
  - Handle `ConnectionClosed` cleanly: log INFO, return 0.
  - Handle `KeyboardInterrupt`: log INFO, return 0.
- Module-level `def main() -> int` parses `argparse.ArgumentParser(prog="aionlslivetiming.cli.jsonl_logger")` with `event_id` (positional), `--output/-o` (positional, required), `--host`, `--max-seconds` (float, optional), then calls `asyncio.run(run(...))`.
- Set `__name__ == "__main__"` guard that calls `raise SystemExit(main())`.
- Log via `get_logger("aionlslivetiming.cli")`.
