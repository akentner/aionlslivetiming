```python
@dataclass(frozen=True, slots=True)
class InitialStateMessage:
    event_pid: ClassVar[int] = 0
    pid: int                       # server export id
    ver: str                       # server schema version
    export_id: str
    track_name: str
    session: SessionInfo
    results: tuple[CarResult, ...] = ()
    best_sectors: tuple[BestSector, ...] = ()
    lts_not_found: bool = False
    raw: Mapping[str, Any] = field(default_factory=dict)
```
Missing `RESULTS` → `results=()`. Missing `BEST` → `best_sectors=()`. Unknown keys stay in `raw`.
